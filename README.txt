Installation Steps:
1. Download and install python 3.x based on your Operating System by following steps on https://www.python.org/downloads/.
2. Unzip file Genome-Sequencing-DBG.zip.
3. Open command prompt/terminal and change the current directory to 'Genome-Sequencing-DBG' which we obtained by unzipping the downloaded file
4. Type the following command:
	python.exe "forest.py"
5. Enter full path of fastq file or path of the dataset.fastq which is sample dataset file in the current directory when propmted for "Enter full file path.."
6. Wait till forest is generated and then follow Menu.
